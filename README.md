# DCF Valuation Model and Financial Data Automation

This project is an Excel-based discounted cash flow valuation model supported by a Python script for financial data collection and organisation.

The Excel model estimates intrinsic value per share for Apple, Microsoft and Alphabet using projected free cash flows, terminal value, enterprise value, equity value and upside/downside analysis. The model includes editable assumptions for WACC, terminal growth, cash, debt, diluted shares and current share price.

The Python script supports the project by pulling and organising financial data for valuation inputs. This project is ongoing, with future improvements planned to convert it into a live valuation tracker and add an integrated WACC calculator.

## Tools Used
- Microsoft Excel
- Python
- Financial data API
- DCF valuation
- WACC
- Equity valuation
